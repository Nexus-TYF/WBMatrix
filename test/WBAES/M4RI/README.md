# WhiteBox AES

An implementation of Chow et al.'s [white-box AES](https://link.springer.com/chapter/10.1007/3-540-36492-7_17) and it's performance test built by [M4RI](https://bitbucket.org/malb/m4ri.git) library.

## Build

```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Run

```
$ ./WBAES
```
